user методы
__init__(заполнить поля) контруктор по умолчанию - все поля должны быть ввиде tuple(Кортеж - не изменный)
form_login(заполнить поля) - конструктор типа формы логина - достает из словаря данные и передает в конструктор по умолчанию
form_register(заполнить поля) - конструктор типа формы логина - достает из словаря данныеи передает в конструктор по умолчанию
db(заполнить поля)- конструктор db вызаваеться только из метода Find_user_by_atr
by_hashkey(заполнить поля) - конструктор кторые по ключи вытяшивает данные из бд

Info(Словарь) - выдает Словарь о доступной ему инфорацией
Create_user(Сообщение) - создаёт нового пользователя в бд с проверкой на не повторность логина почты телефона
Find_user(Сообщение,ключ, зп поля) - провереят пользователя на что бы данные логина и пароля совпадали с полученныеми
Delete_user(Сообщение) - удалить пользователя при удалениие нужно будет передать пароль логин
Update_user(Сообщение) - обновление информации о пользователи
Find_user_by_atr(Сообщение,user) - найти по ключу или логину или паролю илли почте или телефоне
